Example 6 - Flex-grow, Align-Items

With the "flex-grow" property, the outer two boxes are basically labeled the standard size and the 2nd box is made to be twice as large as the standard box size. Please note that "flex-grow" only pertains to width. The reason that the height of the middle item is larger is beause of the 16:9 aspect ratio css rule that is set in Flex Section Global.
